# 七连翻 Online

一个无依赖的网页版联机《七连翻》实现，适合部署在云服务器或局域网内一起玩。

## 运行

需要 Node.js 18 或更高版本。

```bash
node server.js
```

默认监听 `0.0.0.0:3000`。

- 本机访问：`http://localhost:3000`
- 局域网访问：`http://服务器局域网IP:3000`
- 指定端口：`PORT=8080 node server.js`

Windows PowerShell 可以这样指定端口：

```powershell
$env:PORT=8080; node server.js
```

## 云服务器部署

1. 把本目录上传到服务器。
2. 安装 Node.js 18+。
3. 启动：`node server.js`。
4. 在安全组和防火墙中放行对应端口，默认是 `3000`。
5. 浏览器打开 `http://服务器公网IP:3000`。

如果要长期运行，建议使用 `pm2` 或系统服务托管：

```bash
npm install -g pm2
pm2 start server.js --name flip7
pm2 save
```

## 已实现规则

- 数字牌 0-12，数字 N 有 N 张，0 有 1 张。
- 加分牌 `+2/+4/+6/+8/+10`。
- `x2` 牌按官方顺序只翻倍数字牌分数。
- 行动牌：冻结、翻三张、第二机会。
- 重复数字会爆掉；拥有第二机会时可抵消一次。
- 集齐 7 张不同数字自动停牌并获得 `+15`。
- 每轮未爆玩家结算本轮分，先达到 200 且非平局者获胜；最高分平局会加赛。

## 说明

房间状态存放在当前 Node 进程内，重启服务后房间会清空。这个版本适合朋友局和轻量部署；如果需要长期账号、断线保留房间、观战或数据库持久化，可以在现有结构上继续加。
