# RA 太阳神 在线版

零依赖的中文多人网页版，基于 Node.js 原生 HTTP + SSE 实现。

## 启动

```bash
node server.js
```

默认监听 `0.0.0.0:3000`。可用环境变量覆盖：

```bash
PORT=3100 HOST=0.0.0.0 node server.js
```

## 使用

1. 打开页面创建房间。
2. 其他玩家输入房号和昵称加入。
3. 房主开始游戏。

## 说明

- 不需要注册账号。
- 房间状态会写入 `data/rooms.json`，便于重启后恢复。
- 当前实现覆盖基础版《RA》全部牌种、拍卖、灾难、三纪元计分与重开房间。
