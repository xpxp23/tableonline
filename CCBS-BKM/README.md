# 璀璨宝石宝可梦版在线游戏

一个低依赖的网页在线版桌游实现，适合部署在云服务器或局域网内一起玩。服务端只使用 Node.js 内置模块，房间状态保存在内存中，不需要账号、数据库或前端构建步骤。

## 启动

```bash
node server.js
```

或：

```bash
npm start
```

默认监听 `0.0.0.0:3000`，浏览器访问：

```text
http://127.0.0.1:3000/
```

局域网玩家访问主机的局域网 IP，例如：

```text
http://192.168.1.20:3000/
```

也可以指定端口：

```bash
PORT=8080 node server.js
```

PowerShell：

```powershell
$env:PORT=8080; node server.js
```

## 已实现内容

- 2-4 人房间，无注册。
- 90 张中文宝可梦主题牌库：1/2/3 级普通宝可梦、稀有宝可梦、传说/幻之宝可梦。
- 40 个标记：五色精灵球按人数配置，大师球作为万能球。
- 规则动作：拿 3 个不同颜色、拿 2 个同色、保留普通牌、捕捉中央或保留区宝可梦。
- 限制：手牌最多 3 张，精灵球最多 10 个，稀有与传说/幻之不能保留。
- 回合末一次进化、18 点触发终局、完成整轮后按分数结算。
- 浏览器实时同步，不需要刷新页面。

## 牌库说明

公开资料可确认该版本组件规模为 90 张牌和 40 个标记。本实现采用完整可玩的中文宝可梦主题牌库，并把全部卡牌数据集中放在 `game-data.js`。如果你手里有实体桌游的准确卡牌成本与名称，可以直接替换该文件中的 `CARDS` 数据，规则引擎和前端会自动读取。

## 部署注意

- 房间状态在内存中，重启服务会清空房间。
- 如果放到反向代理后面，确保 `/api/events` 的 SSE 不被代理缓冲。
- 本项目没有使用官方宝可梦图片素材，牌面用中文信息、颜色和符号生成，方便自部署。
