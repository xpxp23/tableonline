# Bomb Buster 在线拆弹

一个低依赖的中文网页联机拆弹游戏，适合部署在云服务器或局域网内一起玩。

## 启动

```powershell
node server.js
```

默认监听 `http://0.0.0.0:3000`。如需换端口：

```powershell
$env:PORT=8080
node server.js
```

## 使用

1. 一名玩家创建房间。
2. 其他玩家打开同一网页，输入房间码加入。
3. 房主选择任务并开始。
4. 玩家只能看到其他人的线，自己的线和模块中的未公开线会隐藏。

## 说明

本项目实现了 Bomb Buster 风格的合作推理核心流程：房间联机、玩家线架、初始公开信息、蓝线成对剪断、黄线配对、红线揭示、错误引爆盘、装备解锁、自定义任务参数。

网页内置任务为原创训练/挑战配置；“自定义 / 实体任务卡”可按你手上的任务卡设置参数。
