# 诡镇奇谈：洛夫克拉夫特情书 网页版

这是一个低依赖的中文网页多人版实现，适合部署在云服务器或局域网内一起玩。

## 运行

```powershell
npm start
```

默认监听 `3000` 端口，也可以指定：

```powershell
$env:PORT=3001; npm start
```

打开 `http://服务器地址:端口/`，创建房间后把房间码发给其他玩家即可。

## 说明

- 不需要注册账号。
- 不使用数据库，房间和牌局状态只保存在服务器内存中。
- 不含官方美术、官方排版或说明书原文；卡牌效果与规则提示使用原创中文表述。
- 支持 2 到 6 名玩家。
- 支持清醒/疯狂胜利标记、理智检定、米·戈脑缸、克苏鲁直接胜利等主要规则流程。
